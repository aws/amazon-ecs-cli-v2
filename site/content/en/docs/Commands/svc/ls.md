```bash
$ copilot svc ls
```

## What does it do?

`copilot svc ls` lists all the Copilot services for a particular application.

## What are the flags?

```bash
  -a, --app string   Name of the application.
  -h, --help         help for ls
      --json         Optional. Outputs in JSON format.
      --local        Only show services in the workspace.
```
